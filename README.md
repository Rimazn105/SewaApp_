# SewaApp 🏠

SewaApp adalah aplikasi Android sederhana yang dirancang untuk mengelola data penyewaan dan mencatat riwayat pemasukan secara efisien.

## 🚀 Fitur Utama

Aplikasi ini memiliki beberapa fungsi inti:

* **Tambah Data Baru**: Memungkinkan pengguna untuk menambahkan entitas baru ke dalam sistem (seperti data penyewa atau properti baru).
* **Input Data**: Antarmuka formulir yang mudah digunakan untuk memasukkan informasi detail setiap transaksi atau unit sewa.
* **Pemasukan & Riwayat Data**: Fitur untuk melihat daftar riwayat transaksi yang sudah dilakukan serta total pemasukan yang masuk ke dalam sistem.

## 🛠️ Teknologi yang Digunakan

* **Bahasa**: Kotlin/Java (Sesuaikan dengan yang kamu pakai)
* **IDE**: Android Studio

## 📖 Cara Penggunaan

1.  **Halaman Utama**: Pilih menu tambah untuk memasukkan data barang dan harga barang sewa.
2.  **Input Penyewa**: Isi dengan penyewa dan berapa lama dia menyewa barang.
3.  **Cek Riwayat**: Masuk ke menu Riwayat atau Pemasukan untuk melihat semua data yang telah tersimpan sebelumnya.
